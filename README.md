# 🐕 福といっしょ (Walking App)

愛犬「福」くんの健康管理と散歩記録を、家族みんなで共有するアプリです。

## 🛠 システムの仕組み・使用技術

このアプリがどのように動いているかの記録です。

### 1. 天気情報の取得: OpenWeatherMap
散歩記録をより詳しく残すため、世界中の気象情報がわかるAPI「OpenWeatherMap」を採用しました。

* **採用理由:**
    * 散歩開始地点の正確な「天気・気温・風速」を自動取得するため。
    * 無料枠があり、個人開発での利用に適しているため。
* **動くタイミング:**
    * 「散歩開始」ボタンを押した時だけ通信します。
* **データの扱い:**
    * 取得した天気データは、散歩記録の一部としてデータベース(Firestore)に保存されます。

### 2. その他の構成
* **フロントエンド:** React (CDN版) - 特別なビルド不要で、HTMLファイル1つで動作。
* **バックエンド:** Firebase (Firestore, Authentication, Storage)
* **地図:** Leaflet / OpenStreetMap
